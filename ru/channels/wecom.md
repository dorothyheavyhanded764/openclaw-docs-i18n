# WeCom (WeChat для бизнеса)

Интеграция канала WeCom (企业微信) позволяет пользователям общаться с OpenClaw в **групповых чатах** WeChat для бизнеса, упоминая бота через @. Создайте смарт-бота в режиме API в консоли администратора WeCom, затем настройте **Token** и **Encoding-AESKey** в OpenClaw и укажите URL приёма: `http://<сервер>:<порт>/webhooks/wecom`.

**Полная документация доступна на китайском.** См. [руководство по интеграции WeCom на китайском](../zh/channels/wecom) для пошаговых инструкций (создание бота в консоли WeCom, Token/Encoding-AESKey, настройка канала, URL, проверка домена).

* * *

## Краткая справка

- **Создание бота**: консоль WeCom → 管理工具 → 智能机器人 → 创建机器人 → 手动创建 → API 模式创建; получите и сохраните **Token** и **Encoding-AESKey**.
- **Настройка OpenClaw**: в конфигурации каналов (или в файле конфигурации) укажите **企业微信** с тем же Token и Encoding-AESKey.
- **Указание URL в WeCom**: в форме бота в режиме API укажите URL `http://:<порт>/webhooks/wecom`, затем создайте бота.
- **Проверка**: добавьте бота в группу и упомяните его через @ для потоковых ответов.
