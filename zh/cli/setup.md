# setup

初始化 `~/.openclaw/openclaw.json` 和智能体工作空间。相关内容：

-   入门指南：[Getting started](../start/getting-started.md)
-   向导：[Onboarding](../start/onboarding.md)

## 示例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

通过 setup 运行向导：

```bash
openclaw setup --wizard
```
