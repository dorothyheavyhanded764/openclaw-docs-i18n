# dns

用于广域发现的 DNS 辅助工具（Tailscale + CoreDNS）。目前主要针对 macOS + Homebrew CoreDNS。相关内容：

-   网关发现：[Discovery](../gateway/discovery.md)
-   广域发现配置：[Configuration](../gateway/configuration.md)

## 设置

```bash
openclaw dns setup
openclaw dns setup --apply
```
