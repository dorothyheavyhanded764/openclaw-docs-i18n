# TOOLS.md 模板

技能定义了工具*如何*工作。这个文件是用于*你的*具体细节——你设置中独有的内容。

## 这里放什么

例如：

-   摄像头名称和位置
-   SSH 主机和别名
-   TTS 偏好的语音
-   扬声器/房间名称
-   设备昵称
-   任何环境特定的信息

## 示例

```markdown
### Cameras

- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```

## 为何分开？

技能是共享的。你的设置是你自己的。将它们分开意味着你可以更新技能而不会丢失你的笔记，并且可以共享技能而不会泄露你的基础设施信息。
