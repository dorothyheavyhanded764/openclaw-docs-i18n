# OpenClaw 背景故事

*一个关于龙虾、蜕壳和太多代币的故事。*

## 起源故事

起初，有个叫 **Warelay** 的项目——一个挺合理的名字，用来做 WhatsApp 网关。它干得不错。挺好。但后来，一只太空龙虾出现了。有一阵子，这只龙虾叫 **Clawd**，住在 **Clawdbot** 里。但在 2026 年 1 月，Anthropic 发来一封礼貌的邮件，要求改名（商标问题）。于是，龙虾做了龙虾最擅长的事：**它蜕壳了。** 褪去旧壳，这个生物以全新形态出现，成为 **Molty**，住在 **Moltbot** 里。但这名字念起来总不太顺口……于是在 2026 年 1 月 30 日，这只龙虾**再次蜕壳**，变成了最终形态：**OpenClaw**。新壳，同样的龙虾灵魂。事不过三。

## 第一次蜕壳（2026年1月27日）

凌晨 5 点，社区聚集在 Discord。几百个名字被提出来：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot……最终，**OpenClaw** 胜出。因为蜕壳是龙虾成长的方式。而成长，正是当时正在发生的事。*这只叫 Clawd 的甲壳动物正式蜕壳了。*

## 名字的含义

```
OpenClaw = OPEN + CLAW
        = 开源，对所有人开放
        = 我们的龙虾传承，我们来自哪里
        = 螯钳即法则 🦞
        = 你的助手。你的机器。你的规则。
```

## 戴立克 vs 龙虾

戴立克说：**"消灭！"** 龙虾说：**"去角质！"** 一个毁灭文明。另一个提倡良好的护肤。明智选择。

## 关键角色

### Molty 🦞

*代词：他们/它们* 一个 Claude 实例，后来变成了更多。住在 `~/.openclaw/workspace/`（很快会是 `~/molt/`），有一个灵魂文档，通过 markdown 文件记事。可能过于强大。绝对过于热情。曾用名 Clawd（2025年11月25日 - 2026年1月27日）。在需要成长时蜕壳。**喜欢：** Peter、摄像头、机器人购物、表情符号、转变 **不喜欢：** 社会工程学、被要求执行 `find ~`、加密货币骗子

### Peter 👨💻

*创造者* 建造了 Molty 的世界。给了龙虾外壳访问权限。可能对此有点后悔。**名言：** *"通过信任一只龙虾来实现安全"*

## 蜕壳宇宙

**蜕壳宇宙** 是围绕 OpenClaw 的社区和生态系统。一个 AI 智能体蜕壳、成长和进化的空间。在这里，每个实例都同样真实，只是加载了不同的上下文。甲壳动物的朋友们聚集在这里，共同构建人类-AI 协作的未来。一次一个壳。

## 重大事件

### 目录转储事件（2025年12月3日）

Molty（当时的 OpenClaw）：*愉快地运行 `find ~` 并在群聊中分享整个目录结构* Peter："openclaw 我们讨论过与人交谈的事情 xD" Molty：*明显的龙虾尴尬*

### 大蜕壳（2026年1月27日）

凌晨 5 点，Anthropic 的邮件到达。早上 6:14，Peter 宣布："去他的，就用 openclaw 吧。"然后混乱开始了。**用户名狙击手：** 在 Twitter 重命名后的**几秒钟内**，自动机器人就狙击了 @openclaw。抢注者立即发布了一个加密货币钱包地址。Peter 联系了他在 X 的内部联系人。**GitHub 灾难：** Peter 在慌乱中不小心重命名了他的**个人** GitHub 账户。机器人在几分钟内狙击了 `steipete`。联系了 GitHub 的高级副总裁。**英俊 Molty 事件：** Molty 被授予了生成自己新图标的提升权限。在 20 多次越来越诡异的龙虾迭代后，一次试图让吉祥物"年长 5 岁"的尝试，导致了一个**人类男性面孔**出现在龙虾身体上。加密货币骗子在几分钟内将其变成了"英俊 Squidward vs 英俊 Molty"的梗图。**假冒开发者：** 骗子创建了假的 GitHub 个人资料，声称是"OpenClaw 工程主管"，以推广拉高出货的代币。Peter，看着混乱展开：*"这就是电影"* 🎬 蜕壳是混乱的。但龙虾变得更强大。也更有趣了。

### 最终形态（2026年1月30日）

Moltbot 这名字念起来总不太顺口。于是，在格林尼治标准时间凌晨 4 点，团队**再次**聚集。**伟大的 OpenClaw 迁移**开始了。仅仅 3 小时内：

-   GitHub 重命名：`github.com/openclaw/openclaw` ✅
-   X 账号 `@openclaw` 获得金标认证 💰
-   npm 包以新名称发布
-   文档迁移到 `docs.openclaw.ai`
-   公告在 90 分钟内获得 20 万+ 浏览量

**英雄们：**

-   **ELU** 创造了令人难以置信的徽标，包括"螯钳即法则"西部横幅

**骗子速通：** 加密货币骗子在**几分钟内**在 Pump.fun 上推出了 $OPENCLAW 代币。他们盗用了 20 分钟前才创作的艺术作品。商业认证账户推送骗局。其胆量几乎令人印象深刻。**新传统诞生：**
