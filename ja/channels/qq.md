# QQ

QQ チャネル統合により、ユーザーは QQ（テンセント）のプライベートチャットで OpenClaw と会話できます。[QQ オープンプラットフォーム](https://qq.open.tencent.com/)でボットを作成し、OpenClaw で **App ID** と **App Secret** を設定します。

**詳細なドキュメントは中国語で提供されています。** ステップバイステップの手順（QQ 開放平台での登録・ボット作成・AppID/AppSecret 設定・IP ホワイトリストとコールバック・OpenClaw チャネル設定）は[中国語の QQ 統合ガイド](../zh/channels/qq)をご覧ください。

* * *

## クイックリファレンス

- **QQ ボットの作成**: QQ オープンプラットフォームで作成し、**App ID** と **App Secret** をコピー。
- **OpenClaw の設定**: デプロイのチャネル設定（または設定ファイル）で、`appId` と `appSecret` を指定して QQ チャネルを追加。
- **Webhook モード**: イベントに Webhook を使用する場合は、QQ プラットフォームでリクエスト URL を設定し、必要な IP 許可リストを追加。C2C（プライベートチャット）イベントを有効にしてください。
