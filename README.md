## 项目说明 / Project Description

### 中文

**官网在线阅读：** [www.openclawx.cloud](https://www.openclawx.cloud)

本仓库是 **OpenClaw 官方文档的国际化翻译版本**，面向全球用户提供多语言文档，让不同国家和地区的用户能够用**本国语言**阅读、理解和使用 OpenClaw。

**核心目标：**

- **本土化阅读**：将 OpenClaw 官方文档从英文翻译为多种语言，降低语言门槛，方便非英语用户上手。
- **覆盖多语种**：当前支持英语（en）、中文（zh）、日语（ja）、法语（fr）、俄语（ru）、西班牙语（es）、阿拉伯语（ar），阿拉伯语支持 RTL 布局。
- **与官网一致**：在结构、导航和阅读体验上对齐官方文档站，仅增加语言切换与多语言内容，便于全球用户按自己语言查找文档。

**面向谁：** 希望用母语阅读 OpenClaw 文档的用户；需要为本地社区或客户提供本地化文档的布道师、集成商或社区维护者；关心 OpenClaw 文档可及性与国际化的贡献者与维护者。

### English

**Official docs (online):** [www.openclawx.cloud](https://www.openclawx.cloud)

This repository is the **internationalized translation of the official OpenClaw documentation**. It provides multilingual documentation so that users around the world can read, understand, and use OpenClaw in **their own language**.

**Goals:**

- **Localized reading**: Translate the official OpenClaw docs from English into multiple languages to lower the language barrier and make it easier for non-English users to get started.
- **Multiple languages**: Currently supports English (en), Chinese (zh), Japanese (ja), French (fr), Russian (ru), Spanish (es), and Arabic (ar), with RTL layout for Arabic.
- **Aligned with the official site**: Matches the structure, navigation, and reading experience of the official docs, with added language switching and multilingual content so users can find documentation in their language.

**For whom:** Users who want to read OpenClaw docs in their native language; advocates, integrators, or community maintainers who need localized docs for their region or customers; contributors and maintainers who care about documentation accessibility and internationalization.

### 日本語

**公式オンライン閲覧：** [www.openclawx.cloud](https://www.openclawx.cloud)

このリポジトリは **OpenClaw 公式ドキュメントの国際化翻訳版** です。世界各国のユーザーが **自国語** で OpenClaw を読み、理解し、利用できるよう多言語ドキュメントを提供します。

**主な目的：**

- **現地語での閲覧**：公式 OpenClaw ドキュメントを英語から複数言語に翻訳し、言語の壁を下げ、英語以外のユーザーが使い始めやすくする。
- **多言語対応**：現在、英語（en）、中国語（zh）、日本語（ja）、フランス語（fr）、ロシア語（ru）、スペイン語（es）、アラビア語（ar）をサポート。アラビア語は RTL レイアウトに対応。
- **公式サイトとの一貫性**：構造・ナビゲーション・閲覧体験を公式ドキュメントサイトに合わせ、言語切り替えと多言語コンテンツを追加し、ユーザーが自分の言語でドキュメントを探しやすくする。

**対象者：** 母語で OpenClaw ドキュメントを読みたいユーザー；地域や顧客向けにローカライズされたドキュメントを提供する必要があるエバンジェリスト・インテグレーター・コミュニティメンテナー；ドキュメントのアクセシビリティと国際化を重視するコントリビューターとメンテナー。

### Русский

**Официальная документация (онлайн):** [www.openclawx.cloud](https://www.openclawx.cloud)

Этот репозиторий — **интернационализированный перевод официальной документации OpenClaw**. Он предоставляет многоязычную документацию, чтобы пользователи по всему миру могли читать, понимать и использовать OpenClaw **на своём языке**.

**Цели:**

- **Локализованное чтение**: перевод официальной документации OpenClaw с английского на несколько языков, чтобы снизить языковой барьер и облегчить старт для пользователей, для которых английский не родной.
- **Несколько языков**: поддерживаются английский (en), китайский (zh), японский (ja), французский (fr), русский (ru), испанский (es) и арабский (ar); для арабского — раскладка RTL.
- **Соответствие официальному сайту**: структура, навигация и опыт чтения совпадают с официальной документацией, добавлены переключение языка и многоязычный контент для удобного поиска документации на своём языке.

**Для кого:** пользователи, желающие читать документацию OpenClaw на родном языке; евангелисты, интеграторы или сопровождающие сообщества, которым нужна локализованная документация; контрибьюторы и сопровождающие, заинтересованные в доступности и интернационализации документации.
