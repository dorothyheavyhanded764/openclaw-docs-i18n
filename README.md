## 🌐 What is openclaw-docs-i18n?

This project provides translated versions of the official OpenClaw documentation. It helps users worldwide read and understand OpenClaw content in their own languages. The goal is to make OpenClaw more accessible to non-English speakers by offering multiple language options.

The docs cover key topics such as:
- Clawdbot
- OpenClaw skills
- Tutorials for beginners
- Regional documentation like Chinese and Russian versions

Users will find the translations useful to learn how to use OpenClaw features without struggling with language barriers.
